# watsonx-conversational-ai-assistant
Multi-intent rule-based conversational AI assistant designed using IBM watsonx Assistant.

### Assistant Configuration Export

The file `assistant_configuration_export.json` contains the complete structured dialogue configuration exported from IBM watsonx Assistant.

It includes:

- Intent definitions  
- Dialogue nodes  
- Conditional branching logic  
- Context variable assignments  
- Multi-step workflow structures
